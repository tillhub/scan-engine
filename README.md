# scan-engine
Library which combines Pax, Sunmi &amp; MLKit for scanning engine.

## Usage
Add it in your root build.gradle at the end of repositories:
```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```
Step 2. Add the dependency
```
  dependencies {
    ...
    implementation 'com.github.tillhub:scan-engine:1.0.0'
  }
```
